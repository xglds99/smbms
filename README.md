学校的期末课程设计，项目是烂大街的超市订单管理系统，老师要求每个小组都做这个题目的（我怀疑老师也是上网找的项目题目），而且框架要用SSM + Thymeleaf。
看了网上的一大堆项目还想着能copy一波，结果找到的项目要么是原生web实现，要么就是jsp的还得自己改成Thymeleaf实现。pull了一个网上的项目，改起来更麻烦，不如自己写。
